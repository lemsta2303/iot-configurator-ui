# Frontend - IoT Configuration System

This repository contains the UI of an IoT configuration system developed as part of an engineering thesis.  
The frontend communicates with a separate backend service via REST API.

## Features

- Device listing, filtering and removal (Zigbee / LoRaWAN)
- Adding new Zigbee and LoRaWAN devices via guided workflows
- LoRaWAN gateway management
- Per-device configuration panel
- Attribute include / ignore selection
- Attribute type selection and renaming
- Assignment of data processing functions
- Configuration suggestions (names and processing)
- Saving and updating device configuration
- Client-side validation and clear UI feedback

## Technologies

- React
- TypeScript
- Redux Toolkit
- React Router
- Axios
- Tailwind CSS
- Vite
- ESLint, Prettier, Husky
